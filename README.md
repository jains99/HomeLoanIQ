# HomeLoanIQ

A free, privacy-first home loan calculator for India. No sign-up, no data collection — runs entirely in your browser.

**[Try it live →](https://jains99.github.io/HomeLoanIQ/)**

## Features

- **EMI Breakdown** — see exactly how much of each EMI goes to interest vs principal, month by month
- **Prepayment Planner** — add lump-sum prepayments by month/year and instantly see interest saved and months cut
- **Rate Change Simulator** — model what happens if your bank revises the rate mid-loan (RBI repo rate changes)
- **EMI Optimizer** — enter a target closure date or duration, get the exact EMI needed to hit it
- **Overdraft Loan Comparison** — compare your regular loan against an OD structure where parked savings reduce daily interest
- **Export** — download the full amortization schedule as CSV or print/save as PDF

## Usage

No installation needed. Open `index.html` in any browser, or use the live link above.
